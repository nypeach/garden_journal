ğŸš€ Getting Started

To use the garden journal web forms, you need to start the Flask web server.

Start the Server:

  1. Open Terminal
  2. Navigate to your garden-journal folder: 
    cd ~/path/to/garden-journal
  3. Run the server: 
    python3 src/web_server.py
  4. You'll see a message: Starting server on http://localhost:3000
  5. Open Chrome and navigate to: http://localhost:3000
  6. Keep the server running! Leave the Terminal window open while you use the forms.

When You're Done for the Day:

Press Ctrl+C in the Terminal window to stop the server.
