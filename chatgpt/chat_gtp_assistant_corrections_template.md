# Template

```markdown
**ID**
YOUR-ID-HERE

**TITLE**
Your Title Here

**CATEGORY**
Category Name

**SUB-CATEGORY**
Sub-Category Name

**IF**
The trigger condition goes here. This can be multiple paragraphs and sentences.

**THEN**
The correction response goes here. This can include:
- Multiple paragraphs
- Bullet points
- Code blocks
- Whatever formatting you need

**ANTI-PATTERNS**
Example of what it looked like when wrong
Another example
Third example

**TAGS**
tag1, tag2, tag3
```


## Field Details:

ID - Required - Your custom ID (e.g., DW-RESET-001, PMD-TIMELINE-001)
TITLE - Required - Correction title
CATEGORY - Required - Main category
SUB-CATEGORY - Required - Sub-category
IF - Required - Trigger condition (can be multi-line)
THEN - Required - Correction response (can be multi-line)
ANTI-PATTERNS - Optional - One per line, each becomes a list item
TAGS - Optional - Comma-separated tags

## Notes:

Each section is separated by a blank line
Anti-patterns are split by newlines (one per line)
Tags are split by commas
The "Include footer" checkbox adds the acknowledgment footer automatically when copying
